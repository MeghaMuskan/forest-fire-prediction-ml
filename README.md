# forest-fire-prediction-ml
Machine learning project to predict forest fire risk using regression models (Linear, Ridge, Lasso)

# Forest Fire Prediction using Machine Learning

This project focuses on predicting forest fire risk using machine learning regression models based on environmental factors.

---

## 📊 Dataset
- Algerian Forest Fire Dataset
- Features include:
  - Temperature
  - Humidity
  - Wind Speed
  - Rainfall
  - Other environmental attributes

---

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

---

## 🧠 Models Implemented
- Linear Regression
- Ridge Regression
- Lasso Regression

---

## 🔍 Project Workflow
1. Data Cleaning and Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature Selection  
4. Model Training  
5. Model Evaluation and Comparison  

---

## 📈 Results
- Compared performance of regression models  
- Observed impact of regularization (Ridge & Lasso)  
- Identified key environmental factors affecting forest fire risk
